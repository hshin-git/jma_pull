# jma_news

![ASAS](http://52.193.72.209/jma/latest/ASAS.png?latest)


## 概説
- 気象庁の実況天気図、短期予報解説資料、週間天気予報解説資料の画像化、防災情報のリスト化
- 毎朝、スマホから気象情報をチェックするための個人サイト[JMA News](https://tenki.cf/jma/)で利用
